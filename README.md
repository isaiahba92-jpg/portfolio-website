## Local Development

To run this site locally:

1. Install Jekyll:
```bash
gem install bundler jekyll
```

2. Clone the repository:
```bash
git clone https://github.com/isaiahba92-jpg/portfolio-website.git
cd portfolio-website
```

3. Install dependencies:
```bash
bundle install
```

4. Serve the site:
```bash
bundle exec jekyll serve
```

5. View at `http://localhost:4000`

## Deployment

This site is automatically deployed via GitHub Pages when changes are pushed to the `master` branch.

## Customization

To customize this site:

1. Update `_config.yml` with your information
2. Modify content in `.md` files
3. Add blog posts to `_posts/` directory
4. Customize styles in `_sass/` directory
5. Add images to `assets/images/`
